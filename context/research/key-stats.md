# Forge — Key Facts & Numbers

## Product Stats
- 10 deliberation phases in the workflow
- 5 built-in personas with distinct personalities
- 10 industry-specific persona packs
- 8 session modes (Copywriting, Idea Validation, GTM Strategy, etc.)
- 4 argumentation methodologies (Dialectic, Socratic, Collaborative, Adversarial)
- 5 consensus methods (Unanimous, Supermajority, Majority, Consent, Synthesis)
- 5 export formats (Markdown, PDF, DOCX, HTML, JSON)
- 332 automated tests
- Full conversation memory with rolling summaries
- AI-powered persona generation for any domain
- Multi-language support (English, Hebrew, Mixed)

## Architecture
- Event-driven architecture (not rigid turn-taking)
- Floor manager with priority queues and cooldowns
- SessionKernel: unified engine for CLI and Desktop
- Context scanning: load brand, audience, competitor files automatically

## User Experience
- CLI with interactive menus (@clack/prompts)
- Real-time Ink-based TUI with chat, agent list, status bar
- Breadcrumb navigation through menu flow
- Phase-aware quick replies (1-4 shortcuts)
- Agent suggestion banners during debate
- Session save/resume — pick up where you left off
- Quick Start: goal to debate in under 60 seconds
